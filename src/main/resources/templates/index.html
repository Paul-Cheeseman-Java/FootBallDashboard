Welcome!

This website is a data dashboard that enables you to look at, and investigate, the high level data relating to the Premier and Championship football leagues transfers! The site provides interactive graphs and charts that not only show information, but allow immediate filtering on the dataset through selection of the criteria shown in the graphs, a truly interactive experience!

To use the site click on this link or the 'Data' link in the menu and allow the data to briefly download and the graphs to initialise. Once the graphs have initialised you will be able to see some graphs with a red RESET button, these indicate the interactive graphs which you can click on to filter.

To use the filters if, for example, you would like to see rest of the graphs reflect only winter transfers, select the winter section of the ‘window’ chart. You can apply multiple filters, so if wanted only winter transfers for the 2010-2011 season, select winter, but also the 2010-2011 bar in the in the ‘Value of Seasonal Transfers’ chart. If you apply multiple filters and wish to reset them you can use the main reset button by the menu.
